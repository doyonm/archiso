## I3 keybindings

- $mod + Enter  
  - Open Terminal (Alacritty)
- $mod + e        
  - Exit I3 (Return to lightdm)
- $mod + d        
  - DMenu
  
## Screen Resolution

  - Display mode
    - xrandr
  - Set resolution
    - xrandr --output NAME --mode 1440x900
